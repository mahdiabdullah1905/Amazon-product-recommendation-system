# Amazon Product Recommendation System

This repository contains a machine learning-based product recommendation system for Amazon products. It utilizes collaborative filtering techniques to provide personalized recommendations to users.

## Dataset
The dataset used for this project can be downloaded from Kaggle:
[Amazon Electronics Dataset](https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews)

## Features
- User-based collaborative filtering
- Similarity calculations using cosine similarity
- Personalized product recommendations

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Project Structure
- `User_based_collaborative_filtering.ipynb`: Jupyter Notebook containing the implementation.
- `User_based_collaborative_filtering.py`: Python script version of the implementation.
